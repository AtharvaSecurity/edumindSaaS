**Payments:** Razorpay Integration

## 🚀 Quick Start

```bash
git clone https://github.com/AtharvaSecurity/edumindSaaS.git
cd edumindSaaS
npm install
cd apps/server && npm run dev    # Terminal 1: Backend
cd apps/web && npm run dev       # Terminal 2: Frontend
